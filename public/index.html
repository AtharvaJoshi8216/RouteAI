Integration

One endpoint. That's it.

Replace your provider call. Routing, caching, fallback, and observability handled automatically.

before.sh
❌ Traditional Integration
# Every request. Full price. No fallback.

curl -X POST \
  https://api.openai.com/v1/completions \
  -H "Authorization: Bearer $KEY" \
  -d '{"model":"gpt-4o","messages":[...]}'

# $5.00/million tokens. Always.
after.sh
✅ RouteAI Integration
# Right model. Right price. Auto-fallback.

curl -X POST \
  https://api.routeai.dev/api/ai \
  -H "x-api-key: your-key" \
  -d '{"prompt":"...","mode":"cost"}'

# saved: "$0.0048" | model: "gemini-flash"
# cached: false | latency: 612ms
